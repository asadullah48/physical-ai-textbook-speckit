# Physical AI & Humanoid Robotics Textbook
## Panaversity Hackathon Submission

**Live Demo:** https://frontend-ten-delta-35.vercel.app
**GitHub Repository:** https://github.com/asadullah48/physical-ai-textbook-speckit

## Project Overview
A comprehensive educational platform for learning Physical AI, ROS 2, and Humanoid Robotics with AI-powered assistance.

## Features Implemented
✅ 5 comprehensive learning modules
✅ RAG chatbot powered by Google Gemini AI
✅ Selection-based Q&A feature
✅ User authentication & progress tracking
✅ Interactive code examples with syntax highlighting
✅ Exercise system with difficulty levels
✅ Instructor tools with 13-week course breakdown

## Technology Stack
**Frontend:** Docusaurus, React, TypeScript
**Backend:** FastAPI, Python 3.11+
**AI/ML:** Google Gemini (chat + embeddings)
**Databases:** Neon Serverless Postgres, Qdrant Vector DB
**Deployment:** Vercel

## Architecture
- Specification-first development using Spec-Kit Plus methodology
- RESTful API with JWT authentication
- Vector search for intelligent Q&A
- Responsive design for all devices
